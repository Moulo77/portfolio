Langages de programmation
Java Bonne connaissance
Le Java est un langage asser simple pour apprendre la programmation. En effet, sa syntaxe est proche du C++ mais les principes plus complexes comme les pointeurs ou l'héritage multiple sont simplifiés ou gérer par le langage
C++ Appris et sais utiliser
Le C++ est un langage plus complexe à appréhender que le Java ou Python car il est complet et a de multiples utilisation grâce à ses nombreuses librairies. C'est un langage très utilisé car il à de bonnes performances et un bonne compatibilitée.
Python Appris et sais utiliser
Python est un langage qui utilise beaucoup de librairie et est très utilisé pour l'analyse de données et l'intelligence artificielle. Ce langage, utilisant beaucoup de librairie, est plûtot simple à apprendre il faut en revanche se documenter sur ce qu'on utilise
HTML/CSS Bonne connaissance
Le html ainsi que le css sont souvent des langages facile pour débuter la programmation. À eux seuls ils servent à faire des sites statiques qui, comme ce site, ont pour utilité le partage d'informations.
SQL Appris et sais utiliser
Le SQL est un langage pour questionner des bases de données. Grâce au SQL on peut chercher, modifier ou ajouter les données d'un base de données.
Outils
Git Bonne connaissance
Git est un gestionnaire de version très utilisé. En effet il est très complet et performant, il en va d'un dépot distant à un outil de mise en production automatisé
Docker Appris et sais utiliser
Docker est un outil permettant de créer des containers logiciels. Le principe de docker peut-être difficile à aborder mais plus généralement il sert à embarquer des applications pour qu'elle puisse être utilisé sur n'importe quelle machine
